Batalha Naval em C
Este projeto consiste em uma implementação robusta da Batalha Naval na linguagem C, desenvolvida como trabalho acadêmico. O jogo roda diretamente no terminal, possui inteligência artificial para o oponente e conta com um sistema de persistência de dados em arquivos binários.

Recursos do Projeto
Customização Total: Permite configurar o tamanho do mapa (mínimo de 8x8) e a quantidade de navios (mínimo de 4).

Persistência de Dados: Salva as últimas configurações usadas em historico.bin.

Permite pausar, salvar e sair a qualquer momento para retomar depois via partida_salva.bin.

Modo Solo Avançado: Rodadas dinâmicas onde o jogador e o computador alternam turnos disparando 3 tiros por vez.

Alocação Dinâmica: Todo o tabuleiro é gerenciado dinamicamente em memória por meio de matrizes (int), garantindo bom uso dos recursos de hardware.
